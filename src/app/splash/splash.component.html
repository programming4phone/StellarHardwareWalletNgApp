Welcome to the Stellar hardware wallet! Access to this wallet is secured by possession of the flash drive containing the wallet. All data is stored within files residing on the flash drive so no sensitive financial data is ever stored store locally on the end user's computer. Since this codebase is in the "proof of concept" stage, all access is restricted to the Stellar test network.

Wallet functionality includes:

  • create Create a test account using the Stellar Friendbot and an accompanying passphrase. The newly created account is automatically imported into the wallet.
  • import Import an existing account into the wallet using a passphrase and the account's secret seed.
  • balances Query Stellar account balances.
  • send money Transfer funds (in lumens) between 2 Stellar accounts. A passphrase is needed to perform this operation.
  • transactions Query Stellar account transactions.

Account keys, account number and secret seed, are encrypted to eliminate exposure. Keys are stored within files residing on the flash drive and are never stored locally on the end user's computer. Each imported account is further secured by a user chosen passphrase. Each account can have it's own passphrase or the same passphrase can be used on all accounts imported into the wallet.

The import and send money functions require entry a user chosen passphrase. The balance and transaction query functions access public data within the Stellar network so no passphrase is needed.

An optimal passphrase will consist of a long string of words that is not easily guessed.
For example, chicago weather cold phoenix weather hot

For further details about Stellar accounts see https://www.stellar.org/developers/guides/get-started/create-account.html.
